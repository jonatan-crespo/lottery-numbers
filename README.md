# 🖼️ Lottery-Numbers

Esse repositório tem como objetivo, sortear 6 números de 00 à 99 para que o usuário possa jogar ou utilizar como base no seu jogo da Mega-Sena. O repositório oferece uma plataforma simples onde os usuários podem:<br><br>

## 💻  Funcionalidades
- `Randorizar os Números:` O usuário clica no bottão e recebe 6 números da "sorte".
- `Utilizar:` O usuário pode ou não querer utilizar os números gerados
<br><br>

## 👨‍💻 Como Usar
1. **Clone o Repositório:** `git clone git@github.com:Jonatan-Crespo/lottery-numbers.git`
2. **Acesse o Diretório do Projeto:** `cd lottery-numbers`
3. **Abra o Aplicativo:** Abra o arquivo `index.html` no seu navegador web preferido.
4. **Gere seus números:** Clique no botão *"gerar números"*, para gerar 6 números aleatórios.
5. **Use como quiser:** Utilize os números gerados para fazer a sua fézinha 🙏.
<br><br>
